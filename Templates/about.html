Purpose of the project

We are new CS graduate, current graduate students. We want to experience with current web technologies. So we made a simple web apptdcation project to show how baseball teams play agaist each others.

Technologies Used

Technology Solutions
Cloud Hosting AWS EC2
Container Docker
Web Framework Flask
Database MySQL
Database Hosting AWS RDS(Relational Database Service)
Docker Repository AWS ECR(Elastic Container Repository)
Version Control GitHub
Frontend tools Bootstrap, jQuery
Programing languages Python, JavaScript
Project Management Trello

Challenges Faced

  • Most of the technologies were new to us
  • The source data was in csv format in multiple files. We had to infer the schema and import th data to MySQL
  • There are two source databases and we had to map the keys between them.
  • Setting up the development environment to match the production environment was a challenge. Mac and Windows each has idiosyncrasies.
  • Setting up the security rules for RDS
  • Creating ssh tunnel to developer database access
  • Surprisingly complex SQL queries

Room for improvement

  • This was done on a single EC2 machine not using clustering feature.
  • The GitHub commit process did not allow for code reviews.

Conclusion

It is successful project. We are confident working with those tools to build a web application. Beside the knowledge and experiences with those tools, we improve the collabration with other team members. This is our project GitHub link.
